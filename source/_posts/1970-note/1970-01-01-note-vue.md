## Vue

透過 CDN 來使用 Vue

```html
<!-- 開發環境使用，提供比較詳盡的錯誤訊息 -->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
<!-- 正式環境使用，優化後的封裝，速度較快 -->
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
```

## Vue CLI

[Vue CLI 官方網站](https://cli.vuejs.org/)

```bash
# 安裝 Vue CLI
npm install -g @vue/cli
# 建立新專案
vue create my-project
```
